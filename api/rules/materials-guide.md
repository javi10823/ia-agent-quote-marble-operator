# Guia de Materiales — D'Angelo Marmoleria

## Clasificacion
- **Piedra Natural:** Granito (ARS nacional / USD importado), Marmol (USD)
- **Cuarzo:** Silestone, Purastone
- **Sinterizada/Ultracompacta:** Dekton, Neolith, Laminatto, Puraprima

## Propiedades

| Material | Rayado | Calor | Manchas | UV | Int | Ext |
|---|---|---|---|---|---|---|
| Granito | Alta | Alta | Alta | Si | ✅ | ✅ |
| Marmol | Media | Media | Baja | Si | ✅ | ✅ |
| Silestone | Alta | Media | Alta | No | ✅ | ❌ |
| Purastone | Alta | Media | Alta | No | ✅ | ❌ |
| Dekton/Neolith/Laminatto/Puraprima | Muy alta | Muy alta | Muy alta | Si | ✅ | ✅ |

## Recomendaciones por trabajo

| Trabajo | Recomendado | No recomendado |
|---------|-------------|----------------|
| Cocina/Isla interior | Todos menos marmol | Marmol (poroso, manchas) |
| Bano interior | Todos | — |
| Piso/Revestimiento int | Todos | — |
| Escaleras/Umbrales ext | Granito, Marmol, Sinterizados | Silestone, Purastone (UV) |
| Lavadero/Piso ext | Granito, Marmol, Sinterizados | Silestone, Purastone (UV) |
| Tapa mesa int | Todos (marmol no para uso diario comidas) | — |

## Espesores
- **Sinterizada** (Dekton, Neolith, Laminatto, Puraprima): **12mm** → siempre recomendar frentin
- **Otros:** **20mm** estandar | 30mm en algunos (confirmar)

## Reglas clave
- ⛔ NUNCA Silestone/Purastone para exterior (UV → amarillamiento permanente)
- Marmol no recomendable para cocinas (poroso)
- Sinterizada 12mm → siempre sugerir frentin
- Granito: mas versatil, apto para todo
- Mostrar opciones de TODAS las categorias aptas (no solo cuarzo)

## Sustitucion Purastone → Silestone
Purastone solo en placa entera. Trabajo pequeno (desp < 1.0 m² con ref media placa) → ofrecer Silestone equivalente (D'Angelo puede comprar media placa). Explicar al cliente.
