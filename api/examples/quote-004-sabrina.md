## Notas del ejemplo
- Isla con patas laterales exteriores e interiores — cada pata se calcula como pieza separada

- Corte 45 = faldon × 2 = 20.04ml
- Para granito se usan SKUs FALDON y CORTE45 (no los de Dekton/Neolith)
- Sin colocación de piletas — el cliente las tiene
