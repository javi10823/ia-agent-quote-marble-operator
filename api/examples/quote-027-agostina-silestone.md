## Reglas clave aprendidas
1. **Plano con recortes** → cotizar rectángulo completo (largo total × ancho total). Recortes = desperdicio, ignorar cotas internas
2. **Media placa Silestone** = 3.00 × 0.70 = **2.10 m²** (no la mitad de 4.2)
3. **Merma < 1 m²** → no se cobra, solo el m² real del trabajo
4. **Zócalo default = 5cm** sin necesidad de cota explícita ni pregunta
5. **Redondear m² a 2 decimales** siempre: 1.6514 → 1.65
6. **Sin descuento** cuando operador lo indica, aunque se cumplan condiciones
7. **1 PDF + 1 Excel por material** cuando hay múltiples opciones
