## Reglas clave aprendidas
1. **"FRENTE X cm × Y cm"** → zócalo frontal, pieza de material. No es faldón ni pata
2. **No asumir tipo de ambiente** → usar "Mesada 1", "Mesada 2" si no está indicado
3. **Colocación incluye zócalos** — todo se pega en obra
4. **Flete valor fijo del operador** → $100.000 indicado, no usar catálogo
5. **PUL solo si operador indica** — $30.000 en este caso
6. **Dos cotas de profundidad** (0.60 + 0.42) → dos tramos separados del mismo largo
7. **Sin descuento** aunque se cumplan condiciones — el operador tiene la última palabra
